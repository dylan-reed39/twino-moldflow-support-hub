## Overview

twinO is built as a digital workflow platform for manufacturing scenarios. It gives teams a way to map process steps, keep production-related tasks connected, and maintain workflow rules in a centralized place.

It is particularly suited to organizations using Autodesk Moldflow or handling adjacent manufacturing planning activities. By bringing workflow definition and process support together, twinO helps teams keep routine operations organized and easier to adapt to their own working environment.

- Support for digital workflows across production tasks

## Usage

twinO is meant for defining workflow steps, linking manufacturing activities, and organizing process logic around your production requirements.

Typical workflow:
1. Create a new workflow for a manufacturing process.
2. Add the steps or stages that need to be tracked.
3. Connect the workflow to Autodesk Moldflow-related work where needed.
4. Review and adjust the process as requirements change.

For local development, run the project with the toolchain you use for HTML-based applications and verify the workflow pages in a browser.
